# Punctum Enhancement Proposals

PEP（Punctum Enhancement Proposal）记录 Punctum 的重要设计、实验和流程决策。编号和用途参考 Python PEP，但 Punctum 使用自己的状态和模板。

## 编号规划

- `0000`：PEP 流程。
- `0001`：项目范围和技术方向。
- 后续编号按合入顺序递增，不按主题预留号码。

- 终端 Unicode、宽字符和降级策略。
- 图集、主题与资源 ID 的解析。
