# TEP-0006: 工程结构

## Crate 拆分

三层，依赖单向：app → infra → domain

- **domain**：纯逻辑。零 IO。所有类型、trait、算法。
- **infra**：副作用适配器。crossterm、模拟后端、输入读取器。
- **app**：运行时编排。事件循环、信号管理、终端生命周期。

## 测试分层

domain 纯函数单元测试 → infra 模拟后端集成测试 → 端到端（模拟输入到后端输出断言）。

## 平台与体积

四个平台交叉编译。Release LTO + 体积优化 + 符号剥离，目标 ≤ 8MB 静态链接。

## 当前状态

CI、交叉编译、内存压力测试、dogfooding（netmon 面板）、crates.io 发布全部暂缓。本地调试阶段。
