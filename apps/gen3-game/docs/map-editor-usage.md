# Gen3 地图编辑器使用教程

## 启动

在 `apps/gen3-game` 目录运行：

```powershell
cargo run -p map-editor --release
```

编辑器默认打开 `maps/demo-map.json`。

## 界面

界面分为四个区域：

- 左侧是地图画布。
- 右上是原子素材。每张图片是一个 16x16 素材。
- 底部是组合素材。组合素材可以包含多层原子素材。
- 右下是工具、状态、保存、撤销、重做和帮助。

## 绘制贴图

1. 点击右侧的原子素材。
2. 点击底部的组合素材。
3. 点击“贴图”。
4. 在地图上按住鼠标左键绘制。
5. 在地图上按住鼠标右键擦除贴图。

绘图时使用的是组合素材。选中的原子素材不会直接画到地图上。

## 创建组合素材

1. 在底部选择一个已有组合素材。
2. 在右侧选择一个原子素材。
3. 点击“添加一层”。
4. 编辑器创建一个新的组合素材。
5. 原组合素材保持不变。

新层会叠加在已有层的上方。可以重复添加，不限制层数。

点击“移除顶层”会创建另一个新组合，并删除最后一层。组合素材至少保留一层。

## 删除组合素材

1. 在底部点击要删除的组合素材。
2. 点击右侧的“删除当前组合”，或按 `Delete`。
3. 删除后可以点击“撤销”恢复。

如果组合素材仍被地图上的任意格子使用，编辑器不会删除它。先用其他组合素材改画这些格子，或者使用贴图工具右键擦除，再执行删除。

## 编辑碰撞

碰撞和贴图互不影响。

- 点击“可通行”，再点击地图，可以允许角色通过。
- 点击“阻挡”，再点击地图，可以阻止角色通过。
- 鼠标右键会把碰撞恢复为可通行。

编辑碰撞时，地图会显示绿色和红色遮罩。

## 编辑事件

事件和贴图互不影响。

- 点击“遭遇事件”，再点击地图，可以设置进入格子时触发遭遇。
- 点击“清除事件”，再点击地图，可以删除该格子的事件。
- 鼠标右键也可以清除事件。

编辑事件时，遭遇格会显示黄色遮罩。

## 素材分页

原子素材和组合素材都有“上一页”和“下一页”。

创建第 6 个组合素材后，编辑器会自动显示第 2 页。点击组合素材区的“上一页”可以返回。

鼠标滚轮可以切换原子素材页。

## 保存

点击右下角“保存”。编辑器会写入当前地图 JSON。

游戏和编辑器读取同一份地图文件。保存后重新启动游戏即可加载修改结果。

## 快捷键

| 按键 | 操作 |
| --- | --- |
| `F1` | 打开或关闭帮助 |
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `V` | 切换到贴图工具 |
| `1` | 切换到可通行工具 |
| `2` | 切换到阻挡工具 |
| `3` | 切换到遭遇事件工具 |
| `4` | 切换到清除事件工具 |
| `A` | 添加一层 |
| `D` | 移除顶层 |
| `Delete` | 删除当前组合素材 |
| `PageUp` | 选择上一个原子素材 |
| `PageDown` | 选择下一个原子素材 |

## 数据结构

地图文件保存三套独立数据：

- `visual_cells` 保存组合素材引用。
- `collision_cells` 保存可通行和阻挡。
- `event_cells` 保存遭遇事件。

修改碰撞或事件不会改变贴图。修改贴图也不会改变碰撞或事件。
